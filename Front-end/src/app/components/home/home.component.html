Help Desk Desenvolvido por Elizio neto


Este projeto foi desenvolvido para gerenciar chamados em um cenário simulado de Help Desk, utilizando Angular 12 no frontend,
e um backend robusto em Java 11 e um banco de dados MySQL, rodando em contêineres para facilitar o deploy e a manutenção.
O sistema explora conceitos como SPAs, autenticação e autorização com JWT, além de ser completamente containerizado com Docker e orquestrado com Docker Compose.

O código-fonte está disponível no GitHub, e você é convidado a testar e deixar sugestões.
